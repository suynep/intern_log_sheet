### Technical "Today I Learned" (TIL)

- Flutter builds user interfaces using widgets as the fundamental building blocks, where everything, from layout structures to visual elements, is composed of nested widgets.
- Material widgets provide pre-designed UI components that follow Google's Material Design principles, enabling rapid development of visually consistent applications.
- Flutter layouts are created by composing widgets like Row, Column, Container, and Expanded, allowing flexible arrangement and responsive design without traditional layout files.
- The widget tree architecture encourages modular design, making it easier to separate UI components into reusable parts.
